# Alloco Theme Page

Static frontend package split for GitHub/Vercel deployment.

## Structure

- `index.html` — main page
- `css/styles.css` — all styles
- `js/app.js` — main interaction logic
- `js/launcher.js` — ETP launcher wizard logic
- `assets/` — extracted image assets

## Deploy on Vercel

1. Push this folder to GitHub.
2. Import the GitHub repo in Vercel.
3. Use default static site settings.
4. No build command is required.
